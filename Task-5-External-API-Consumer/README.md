\# Task 5 - External API Consumer



\## Project Overview



This project was completed as part of the Growfinix Python Internship.



The application accepts a city name from the user, retrieves its geographical coordinates using the Open-Meteo Geocoding API, fetches live weather information from the Open-Meteo Weather API, processes the JSON response, and displays a user-friendly weather report.



\---



\## Objectives



\- Accept user input

\- Connect to public REST APIs

\- Send HTTP GET requests

\- Parse JSON responses

\- Display live weather information

\- Save JSON responses

\- Generate a weather report

\- Handle API and network errors



\---



\## Technologies Used



\- Python

\- Requests

\- JSON

\- Open-Meteo Geocoding API

\- Open-Meteo Weather API
